# 📊 Aprendendo Consultas SQL - Banco Contoso

Este repositório foi criado para documentar e organizar meus estudos em banco de dados utilizando a linguagem **SQL** no ambiente **SSMS (SQL Server Management Studio)**. 

Os exercícios e desafios práticos foram aplicados sobre o banco de dados **Contoso** (da Microsoft), que simula um cenário real de grande porte de varejo, e-commerce e inventário.

---

## 📂 Estrutura do Repositório

O projeto está organizado da seguinte forma:

* `📁 Fundamentos/`: Pasta contendo scripts individuais focados na fixação dos pilares da linguagem SQL.
* `📁 Exercícios/` : Pasta contendo scripts com diversos exercícos de fixação dos conceitos abordados.

---

## 🛠️ Conceitos e Cláusulas Praticadas

Durante o desenvolvimento deste projeto, explorei e dominei os seguintes conceitos de manipulação de dados:

* **Filtragem de Dados:** Uso estratégico de `WHERE`, além de operadores lógicos obrigatórios (`AND`) e alternativos (`OR`) prestando atenção na ordem de precedência.
* **Limitação e Seleção:** Aplicação de `TOP` e `TOP PERCENT` para amostragem de dados e análise de extremos.
* **Agregações:** Uso de funções estatísticas e matemáticas como `SUM()`, `AVG()`, e `COUNT()`.
* **Agrupamentos Dinâmicos:** Consolidação de relatórios usando `GROUP BY` e filtros avançados pós-agrupamento com `HAVING`.
* **Organização de Código:** Formatação de visualização através de `ORDER BY (ASC/DESC)` e uso de aliases (`AS`) para renomeação limpa de colunas.
* **Relacionamento de Tabelas (JOINs):** Uso dos diferentes tipos de `JOIN ` para cruzar tabelas de dimensão (como `DimProduct` e `DimCustomer`) com tabelas de fatos (`FactSales`), permitindo análises ricas e consolidadas a partir de chaves primárias e estrangeiras.
